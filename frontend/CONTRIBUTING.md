# Contributing

## Creating a New Chapter

1. **Copy the Template**:
   Create a new file in `docs/modules/module-X-name/` using the format below.

2. **Metadata**:
   Ensure all frontmatter fields are present.

3. **Images**:
   Save images to `static/img/modules/module-X/`.

### Chapter Template

```markdown
---
title: <Chapter Title>
description: <Short description for SEO>
sidebar_position: <Order Number>
tags: [ros2, python, tutorial]
draft: false
---

# <Chapter Title>

## Introduction
Briefly explain what this chapter covers.

## Core Concepts
...

## Hands-on Lab
...

### Prerequisites
- [ ] Completed previous chapter
- [ ] Hardware ready

### Steps
1. Step 1
2. Step 2
```
